== Description ==

Add Maxmail subscription forms to your website. 

To use this plugin you must have a Maxmail account 
- http://www.maxmailhq.com
- Sign up for a free account now


== Installation ==

1. Upload the contents of this zip to the /wp-content/plugins/ directory, e.g. you should end up with a file: /wordpress/wp-content/plugins/maxmail/maxmail.php
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. In the Maxmail plugin's settings, enter your credentials from Maxmail:
	* Maxmail Email     - This is the email address you use to log in to Maxmail
	* Maxmail API Key   - Find this in your Maxmail login under Settings -> API Credentials
4. Click Save, and you will see your Maxmail lists. If you do not have any, create one in Maxmail. 
5. Select the list you wish your visitors to subscribe to.
6. Place [maxmail] in your page to display the form.
7. Try it out, enter your information and hit Subscribe, you will be added to the list in Maxmail.
